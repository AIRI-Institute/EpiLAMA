# EpiLAMA
This a repository accompanying the "Cytokine-Driven Immunogenicity Prediction: Integrating HLA Binding and Cytokine Induction" paper. This paper introduces a suite of cytokine release prediction models EpiLAMA-IL and an immunogenicity prediction model EpiLAMA-IM.

For details on how to use either EpiLAMA-IL or EpiLAMA-IM see their respective README files: [EpiLAMA-IL README](EpiLAMA-IL/README.md) and [EpiLAMA-IM README](EpiLAMA-IM/README.md).
